## Setting up backend with Node.js

**npm init -y**

### installing dependencies

**npm install --save-dev babel-cli babel-preset-env**

create file .babelrc :

{
  "presets": ["env"]
}

**npm install babel-watch --save-dev**
**npm install express** 

**npm install graphql graphql-tools apollo-server-express**
